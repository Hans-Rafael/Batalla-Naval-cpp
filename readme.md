
# Batalla Naval

Batalla Naval es un juego de adivinar barcos en línea desarrollado como parte del segundo parcial de la materia. El proyecto incluye los siguientes elementos:

## Funcionalidades

- Estructuras condicionales
- Estructuras de repetición
- Estructuras de datos (arreglos, matrices)
- Funciones

## Descripción del funcionamiento

Batalla Naval es un juego de adivinar barcos en línea donde dos jugadores intentan hundir los barcos del otro jugando turnos. El juego se juega en una matriz de 3x3.

El programa utiliza estructuras condicionales y estructuras de repetición para controlar el flujo del juego y las interacciones del usuario. Las estructuras de datos, como arreglos y matrices, se utilizan para almacenar la ubicación de los barcos y los disparos realizados por los jugadores. Las funciones se utilizan para manejar las diferentes acciones del juego, como colocar barcos, realizar disparos y verificar si un barco ha sido hundido.

## Decisiones tomadas durante el desarrollo

Durante el desarrollo del proyecto, se tomaron las siguientes decisiones:

- Se decidió utilizar un enfoque modular, dividir el código en funciones y módulos para facilitar la comprensión y mantenimiento del programa.
- Se utilizó una matriz de 3x3 para representar el tablero de juego, lo que simplificó la implementación de las reglas del juego.
- Se implementaron funciones para verificar si un barco ha sido hundido y para determinar si un jugador ha ganado.

## Instalación

Para jugar a Batalla Naval, necesitas tener instalado un compilador C++ y un entorno de desarrollo integrado (IDE) para compilar y ejecutar el código fuente.

## Uso

1. Compila el código fuente utilizando un compilador C++.
2. Ejecuta el programa.
3. Elige entre jugar contra la computadora o contra otra persona en la misma computadora.
4. En el modo contra la computadora, el juego alternará turnos hasta que uno de los dos se quede sin barcos.
5. En el modo contra otra persona, cada jugador coloca sus barcos por separado y luego se alternan los turnos. El tablero enemigo se muestra ocultando los barcos.
